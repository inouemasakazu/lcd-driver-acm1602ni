## 概要
「ACM1602NI-FLW-FBW-M01」の機能・動作検証実施を目的に作成していたプロジェクトです。  

このプロジェクトは、「lcd-driver-acm1602ni」とは異なるリポジトリとして管理していましたが統合致しました。  

統合理由は、  
- 同じLCDデバイスを対象としたプロジェクトであるため
- 管理体制整理のため


現在のAPI仕様とは異なるため、互換性はありません。  
また、基本的には今後の更新もないと思います。
